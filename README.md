# 使用方法：
1.需要去登录个人所得税app查看，每个月税前收入、每个月社保合计、以及当前已申报纳税额；
2.需要自己计算专项附加扣除数，就是子女教育、赡养老人、房贷、房租等；
3.支持奖金单独或合并计算；
